Projeto 02 - Site com formulário de cadastro de usuários associado

  • Crie um site que tenha uma página principal explicando algum serviço específico. Para acessar o serviço será necessário fazer um cadastro.

  • Ao solicitar acesso, o usuário deve ser direcionado para uma segunda página contendo um formulário de cadastro que conterá campos como nome, cpf, e-mail, telefone, perfil do github, etc.

  • Ao clicar no botão cadastrar, a página deverá fazer a validação dos dados inseridos no formulário e apresentar uma mensagem informando se o formulário está preenchido corretamente ou não (CPF válido, Nome não pode estar em branco, e-mail em formato válido, telefone em formato válido no Brasil, perfil do github em formato de URL).
